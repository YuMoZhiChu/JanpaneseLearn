
# 语气词

### んです、のです

**（動詞普通形）＋んですが・のですが**
**（い形容詞）＋んですが・のですが**
**（な形容詞）＋なんですが・なのですが**
**（名詞）＋なんですが・なのですが**
- 表示强调，委婉，客气的语气，具体看语境（翻译可略过，再加上一个语气即可

### どんなに……だろう

**どんなに+動詞普通形・な形容詞・い形容詞+だろう**

```
どんなにうれしいだろう。
该多么开心呀
```
- 强烈的表达修饰的内容的感情

### ものだ

**（動詞普通形）＋ものだ**
**（い形容詞）＋ものだ**
**（な形容詞）＋なものだ**

```
知らない町を旅して、色々な人々に会うのは楽しいものだ。
在陌生的城市旅行，与形形色色的人见面实在是一件愉快的事情。
```

- 表达对事物的感叹，惊讶的语气。相当于，实在是……啊，太……了

# 顺接

### たら

#### 如果…就（假定顺接

**（動詞た形）＋ら**
**い形容詞（~~い~~）＋かったら**
**な形容詞・名詞＋だったら**
```markdown
雨が降ったら、バーベキューは中止です。
雨だったら、バーベキューは中止です。
如果下雨的话，就中止野餐。
```
- 可以在句首加上```もし```来强调语气

### それに

#### 再加上、并且（第三个原因的顺接

```
晴れたし、それに風もない
天很晴朗而且没有风。

この料理まずいし、それに高い
这个饭又难吃又贵。
```

### それで

#### 因此，所以

```
彼は予備校へ通った。それで成績がよくなった。
他在补习学校学习，所以成绩好了。
```

#### 那么，后来。

```
それでどうした？
后来怎样了？
```

### に従って

#### 随着……，变得……

**（動詞辞書形）＋に従って**

```
年を取るに従って、体力が落ちて来ました。
随着年龄增长，体力下降了。
```
- 前后都应该是表示**变化**的语句

### につれて

#### 随着……，逐渐……

**動詞辞書形・名詞（変化を表すもの）＋につれて**

```
達雄は成長するにつれて、頼もしくなりました。
随着达雄的成长，变得可以依靠了。
```

- 前后都应该是表示**变化**的语句，后面一句，**不能出现意志性**的表达

### 以上

#### 既然……就……

**普通形（な形容詞・名詞　～である）＋以上**

```
引き受けた以上、責任を持つ
既然承担下来，就要负责。
```

- 前面表示某种既定的事实，引申出来的结果。一般前面是动词的`た形`，表示既然发生了……，就……

### ない限り

#### 只要不……就……

**（動詞ない形）＋ない限り**

```markdown
一生懸命に勉強しない限り、日本語は上達しません。
只要不十分的努力，日语就不能很好。
```

- 表示事物在前项条件没有达成的情况下，后项的无法实现。

### さえば

#### 只要……就……

**（名詞・動詞ます形）＋さえ＋動詞仮定形**

```markdown
考えさえすれば、誰にでもできる問題です。
只要考虑一下，谁都能完成的问题。

お金さえあれば、何でも買える。
只要有钱什么都能买。
```

- 表示唯一的条件。除此之外什么都不需要。只要满足了前项即可

### しかも

#### 而且

```markdown
日は暮れて,しかも雨まで降ってきた
天黑了,而且雨也下起来了.
```

# 逆接

### でも

#### 但是

```markdown
あの人に何度も手紙を出してみたわ。でも、一度も返事をくれなかったの。
我给他去过好多次信。可是，他一次也没给我回信。
```

#### 即使……也、不管……都

**（動詞て形）＋も**
**い形容詞（~~い~~）＋くても**
**な形容詞・名詞＋でも**
```markdown
雨が降っても、バーベキューをしましょう。
即使下雨，也要野餐。

いくら寒くても、スカートを履きたい。
不管有多冷，也想穿短裙
```
- 可以在句首加上```いくら```，`どんなに`来强调语气

#### 尽管说是……，却

**動詞・い形容詞・な形容詞（普通形）　＋とは言っても**
**名詞＋とは言っても**
```
いくら仕事とは言っても、健康に気をつけてください。
虽然是工作，也要注意健康。
```
- 对其不够的地方用后项进行补充说明。

### だって

#### 即使……也、不管……都（でも　口语化

**（動詞て形）＋も**
**い形容詞（~~い~~）＋くても**
**な形容詞・名詞＋でも**
```markdown
雨が降っても、バーベキューをしましょう。
即使下雨，也要野餐。

いくら寒くても、スカートを履きたい。
不管有多冷，也想穿短裙
```
- 可以在句首加上```いくら```，`どんなに`来强调语气


### けど

#### 但是，其实

```markdown
あの人は丈夫そうにみえる。けど、よく病気をする。
那个人看起来很壮实，但（实际上）常常生病。
```

### のに

#### 明明……，却……

**動詞普通形・イ形容詞　＋のに**
**名詞・な形容詞　＋のに**

```markdown
今日は日曜日なのに、出勤しなけばなりません。
今天明明是周日，但是要上班

約束したのに、達雄さんはきてくれませんでした。
明明约好了，达雄还是没有来。·
```

### にしても

#### 明明……，却……

**動詞・イ形容詞・な形容詞　（普通形）　＋にしても**
**名詞　＋にしても**

```markdown
わたしにしても困る
就是我也不好办
```
- 接续，是普通形都可以，名词有原本
- 表示，承认前项既成事实，后项提出说话人的评价，判断，难以接受的心情等。

### しかも

#### 但是（逆接

```markdown
注意を受け、しかも直そうとしない。
被提醒了，而且不打算改正。
```

### ところが

#### 可是，但是

```markdown
達雄さん、この服、高かったでしょう。
ところが、やすかったんだ。

达雄君，这件衣服，很贵吧。
可是，很便宜哦
```
- 前项所推测或期待的内容与既成事实的后项不一致。
- 后面接事实，一般是过去式。

# 表示原因

### て

**動詞て形、**
**イ形容詞て形、**
**な形容詞て形、**
**名詞＋で、**
```markdown
達雄さんの話を聞いて、がっかりしました。
听了达雄君的话，很失望。

都合が悪くて、行けない。
身体不舒服，去不了了。

綺麗で、好きです。
因为很漂亮，所以很喜欢。

病気で会社を休みました。
生病了就休假了。
```
- 句末不能加，说话人的意志或者要求做某事
- - 意志：我想做什么
- - 要求：能帮我做什么么`てください`

### から

**（動詞・い形容詞・な形容詞・名詞　普通形・丁寧形）＋から**
```markdown
危ないから、この乗り物には乗るな。
很危险，不要做这种交通工具。
```
- 这种表达，句末常会出现，命令形，禁止形，`なければならない`，`てください`，这些表达说话人意志或者**要求**对方做什么的表现
- 语气比较僵硬，不太适合用于拒绝，**请求**（日本人喜欢委婉，这里要求，和请求，还是不同的
- 在这里，`丁寧形`和`普通形`都是ok的
```
ありませんから・ないから
便利ですから・便利から
雨でしたから・雨だったから
```

### ので

**（動詞・い形容詞　普通形）＋ので**
**（な形容詞・名詞）＋な＋ので**
```markdown
体調が悪いので、お先に失礼します。
身体不舒服，就先溜了。
```
- 因为`な形容詞・名詞`的普通形，都是后面接`だ`,在这里呢，用`な`
- 语气比较委婉，通常用于拒绝，请求
```
ないので
便利なから
雨だったので
雨なので
```

# 变成

### なる

**い形容詞（~~い~~）＋くなります**
**な形容詞＋になります**
**名詞＋になります**
```markdown
熱くなる。
变热

きれいになる。
变得漂亮

テレビの音が大きくなりました。
电视的声音变大了。
```
- 这个表示的是，人或事物的状态，自身发生了变化，变成……


### する

**い形容詞（~~い~~）＋くします**
**な形容詞＋にします**
**名詞＋にします**
```markdown
今日は部屋をきれいにします。
今天要把房间打扫干净。

テレビの音を大きくしますよ。
电视的声音调大点了哦。

お昼ご飯はパンにします。
中餐决定是面包了！

```
- `する`在特定的情况下，可以表示 **将……变得……**，或者理解成**让……是……**，所以名词的接续，也有决定的意思


### ようになりました

**動詞辞書形＋ようになりました**
```markdown
新しい地下鉄で通勤できるようなりました。
因为新的地铁，所以能够通勤了。
```

**動詞ない形＋なくなりました**
```markdown
少し太って、好きな服が着られなくなりました。
稍稍变胖了，喜欢的衣服穿不了了。
```

- 表示能力，状况，习惯的变化，一般使用可能动词比较合适
- 不能接表示变化的动词（如果这类动词，直接用即可
- 一般是```なりました```，因为已经变化了

### 気分になる

**修飾＋気分になる*

```markdown
この町に来ると何となく懐かしい気分になる。
我一来到这个城市就有种怀念的感觉。
```

- 接续，前面就是修饰`気分`的修饰语
- 强调心里的状态发生了变化，相当于汉语的 **变得有……的气氛，变得有……的心情**

# 为了（不）能够实现……这一目标

### ように

**（動詞辞書形）＋ように、**
```markdown
仕事がうまくいくように、毎朝新聞を読んでいます。
为了保证能很好的工作，每天早上都读新闻。
```

**（動詞ない形）＋ないように、**
```markdown
先生から聞いたことを忘れないようにメモをとります。
为了记住老师讲的东西，做笔记
```

- 前面用非意志性的动词（前面应该是比较客观的事物，不能表示为了一些个人想法的目标，比如进大学

### ために

**（名詞）＋の＋ように、**
```markdown
将来のために、お金を貯めています。
为了将来，一直在存钱。
```

**（動詞辞書形）＋ために、**
```markdown
大学に行くために勉強しています。
为了上大学努力学习。
```

- 前面用意志性的动词（非常主观的，比如我想上大学，这不是一个客观的现象

# 不可以……/必须做……

**（動詞て形）＋はならない**

- 根据社会常识，一般规律的判断。表否定时，表示必须做某事

# 必须做……

### なければなりません

**（動詞ない形）＋なければなりません**
```markdown
ボールペンで書かなければなりません。
必须用圆珠笔写。
```
- 语气很重，不管行为者的意志如何，都要做
- ```なきゃ```简写

### ないといけません

**（動詞ない形）＋ないといけません**
```markdown
ボールペンで書かないといけません。
不用圆珠笔写是不行的。
```
- 语气中等，从行为者的意志出发，类似于，不怎么样就不可以的哦，的劝说
- ```ないと```简写

# 不能再……，无法再……

- 表示由于某种情况，动作主体在心情上无法控制，不能再保持原来的状态

# 授受动词

## 给（为）我做……
他人 → **我**

### くれます

**（誰）は＋（私に）＋（名詞）＋を＋くれます**
**（誰）は＋（私に）＋（動詞て形）＋くれます**

```markdown
達雄さんは、私に、ケーキをくれました。
达雄给了我蛋糕。

達雄さんは、私に、電話番号を教えてくれました。
达雄告诉了我电话号码
```
- 表达受到恩惠，表达感谢。语气要十分委婉
- ```私に```可以省略，因为```くれる```出现了，一定是我受到了恩惠

### くださいます

```markdown
達雄さんのお母さんは浴衣の着方を教えてくださいました。
达雄的母亲教给了我和服的穿着方法。
```
- `くれます` ，对上级的模式

### くださいませんか

```markdown
印刷機の使い方を教えてくださいませんか。
能告诉我印刷器的使用方法么？
```
- `くださいます` ，的**疑问态**，一般用于问句，能给我做什么么？

## 我从……得到……（恩惠）
我 ←**获得恩惠**← 他人

### もらいます

**（私）は＋誰に・から＋（名詞）＋を＋もらいます**
**（私）は＋誰に・から＋（動詞て形）＋もらいます**

```markdown
私は、紀子ちゃんに、電話番号を教えてもらいました。
我向纪子要了电话号码。
```
- 表达受到恩惠，表达感谢。语气要十分委婉
- 这里的```に```可以替换为```から```

### いただきます

```markdown
課長にビジネス文書の書き方を教えていただきました。
我向课长请教了商务文书的书写方法。
```
- `もらいます` ，对上级的模式

### いただけませんか

```markdown
課長、ビジネス文書の書き方を教えていただけませんか。
课长，能教我商务文书的书写方法吗？
```
- `いただきます` ，的**可能态**，一般用于问句，能给我做什么么？

## 我为……做……

###  あげます

**（私）は＋誰に＋（名詞）＋を＋あげます**
**（私）は＋誰に＋（動詞て形）＋あげます**

```markdown
私は、達雄さんに、ケーキを作ってあげました。
我为达雄做蛋糕
```
- 表达**给予恩惠**，不能对关系生疏的，地位高的人用

### やります

```markdown
八チに餌を与えてやりました。
```
