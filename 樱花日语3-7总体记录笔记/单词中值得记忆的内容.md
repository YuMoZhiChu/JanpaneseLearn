
# 特殊用途单词

## 中

### じゅう

```markdown
世界中一番
世界第一
```
- 这个读音，表示，全部，所有，相当于**全……之中**

### ちゅう

```
テストちゅう
考试中
```
- 这个读音，表示动作或者状态正在进行中，相当于**在……进行之中**


### 途中

#### 在/正在……半途上


**（動詞て形＋いる）＋途中で**
**（動詞辞書形）＋途中で**
**（名詞の）＋途中で**
```markdown
実は、途中で事故があって、荷物が遅れてしまいました。
实际上，中途遭遇了事故，货物延期了。
```
- 这里的接续，实际上是，动词和名词的一类接续，但是这个不同的是，表示途中，只能用`動詞辞書形`或者`動詞て形＋いる`

## 場合、時

### 如果……的话

**（動詞ない形）＋場合は・時は**
**＋場合は・時は**

```
予定変更の場合は、早めに電話します。
预约变化的话，尽快给电话
```

- 本质，是名词的接续方法，因为本来就是名词

## 訳

### 理应……

**動詞・い形容詞　普通形＋訳だ**
**な形容詞（～な・～である）＋訳だ**
**名詞（～の・～である）＋訳だ**

```
桜ちゃんは上海で育ったから、中国語が上手なわけだ。
小樱在上海长大，中文应该很好才对。
```

- 表达事物在前项条件下，做出的推断。

## ぬ

### ～ない

```
達雄さんはわたしになくてはならぬ存在です。
达雄对我来说，是不可缺少的存在。
```

- 相当于`ない`，有时后面接名词时，可以用`ぬ`代替


## ず

### ～ないで・～なくて

```
桜ちゃんの本当の気持ちも分からず、彼女を傷つけてしまいました。
我不知道小樱的真实想法，伤害了她。

美味しず
不好吃
```

- 相当于`ないで・なくて`，当接尾词时，可以用`ず`代替

## 首

### 首にする

```
命令を服従しないため、社長は彼を首にした。
由于不服从命令，社长解雇了他。
```

- 辞职，被辞退用被动

### 首を縦に振る

```
命令を服従しないため、社長は彼を首にした。
由于不服从命令，社长解雇了他。
```

- 点头，答应，同意。

### 首を横に振る

```
本社は私たちの計画に首を横に振った。
总公司没同意我们的计划。
```

- 摇头，不论，不赞成。

## ものを言う

### ものも言わずに

```
桜ちゃんは、ものも言わずに去ってしまいました。
小樱一言不发地走了。
```
- 表示什么也不说

### ものを言う

```
困難に陥った時、経験がものを言います。
面对困难的时候，经验
```
- 表示起决定作用

# 特殊的一词多意

## 口にする

#### 说到，提到

```
人の悪口を軽々しく口してはいけません。
不要轻易说别人的坏话。
```

#### 吃喝

```
その料理は今までに口にしたことの味がしました。
那个菜有至今为止吃过的味道。（以前没吃过
```

## 手にする

### 手にして

#### 拿在手里

```
中田課長が企画書を手にして会議室に入ってきました。
中田科长拿着企划书走进会议室。
```

### 手にした

#### 弄到手，到手

```
会社の授賞式でトロプィーを手にした達雄さんはうれしく涙を流しました。
在公司的颁奖仪式上拿到奖杯的达雄高兴地流下了眼泪。
```

## 当てる

#### 撞到，贴近，触膜

```
今朝急いでいたので、新車を壁に当ててしまいました。
因为今天早上很着急，所以新车撞到了墙上。
```

#### 晒到，照射到

```
十分日に当てた布団に寝るのは、気持ちがいいです。
睡在晒了充分太阳照过的被子里，心情很好。
```

#### 安排

```
僕は毎日二時間を中国語の勉強に当てています。
我每天花两个小时学习中文。
```

#### 被……叫

```
中国語教室で、先生に当てられたときは緊張しました。
在中文教室里，被老师叫到的时候我很紧张。
```

#### 猜

```
桜ちゃん、手の中に何が入っているか当てて見て。
樱酱，猜猜看手上有什么。
```

#### 当作

```
仕事をする時は、お客さんの立場に自分を当てて考えることが大切です。
工作的时候，把自己放在客人的立场上思考是很重要的。
```

## 断る

#### 预先通知；事前请示；事先说好

```
誰にも断らずに入ってくる。
未经请示就进来了。
```

#### 谢绝；拒绝；推辞

```
きっぱり断る。
断然拒绝。
```

## これだけ

#### 表程度、范围。

```
 これだけの費用がかかる。
需要这么多费用。
```

#### 特定，尤其。

```
これだけは忘れるな。
可别忘了啊。
```

#### 强调分量。

```
これだけ言っても聞かない。
说了这么多也不听。
```


# 接尾词

## すぎ

**動詞ます形＋すぎ**
**い形容詞（~~い~~）＋すぎ**
**な形容詞＋すぎ**

```
夕べ、お酒を飲み過ぎました。
傍晚，喝酒喝多了

暑すぎ
太热了

贅沢過ぎ
太奢侈了
```
- 表示行为或者状态，超过了范围。

## やすい、にくい

**動詞ます形＋やすい**
**動詞ます形＋にくい**

```
薄いシャツは乾きやすいです。
薄的衣服容易干

厚いシャツは乾きにくいです。
厚的衣服不容易干
```
- 表示某种事情容易做，或者不容易做

## 済む

**動詞て形＋済む**
**名詞～で＋済む**

```
「遅刻は謝って済むことではありません」と先生に言われました。
“迟到不是道个歉就可以解决的事情”老师对我这么说。

それだけで済むと思うのか。
你以为那样就过得去吗？
```
- 表示**采样某种方式就可以解决问题**。相当于汉语的，做……就可以了


## ～込む

### ~得厉害

**動詞ます形＋込む**

```
咳き込む
不住地咳嗽

寝込む
入睡，熟睡

勢い込む
振奋，奋起，鼓起干劲，满怀信心

追い込む
紧排，挤排，移前
```
- 强调状态，程度

### 一个劲的做~

**動詞ます形＋込む**

```
煮込む
煮熟，煮透

泊まり込む
（因事回不了家而）住宿，投宿，住下，过夜

考え込む
沉思，苦想。

磨き込む
磨光擦亮
```

- 表示很长时间做某事，到动作结束为止重复做一个动作
