
# 但是...（逆接

### でも
```markdown
今からでもおそくはありません。
即使从现在起也不晚。
```

### けど
```markdown
あの人は丈夫そうにみえる。けど、よく病気をする。
那个人看起来很壮实，但（实际上）常常生病。
```

# 想要（某物）

### ほしい

**（名詞）＋が＋ほしいです**
```markdown
私は、パソコンがほしいです。
我想要电脑。
```
- **名詞 + が**：这里的名词用```が```来提示

# 想做（某事）

### たい

**（動詞ます形）＋たいです**
```markdown
私は、すしが(を)たべたいです。
我想吃寿司（が 表示强调语气）
```

# 做……吧。

### ましょうか。

**（動詞ます形）＋ましょうか**
```markdown
明日も来ましょうか。
ええ、１０時に来てください。

我明天也来吧？
恩，请10点钟过来。

傘を貨しましょうか。
すみません。お願いします。

我把伞借给你吧。
谢谢，我借一下。

お腹がペコペコだよ。
じゃ、ご飯を食べに行きましょか。

肚子饿了啦。
那，我们一起去吃饭吧。

```
- 注意3种不同的使用场景，可以用于自身，或者一起的提议（一定要有自己

# 请……。

### てください

**（動詞て形）＋ください**
```markdown
ちゅっと待ってください。
请稍等一下下。

駅の読み方を教えてください。
请告诉我车站的读法。
```
- **すみませんが、…… +（動詞て形）＋ください**：在句首加 ```すみませんが``` 会显得更加委婉

# 请不要……。

### ないでください

**（動詞ない形）＋ないでください**
```markdown
このパソコンは、使わなくてください。
请不要使用这台电脑。
```

# 可以做……。

### てもいいです

**（動詞て形）＋もいいです**
```markdown
パソコンを使ってもいいです。
可以使用电脑哦。

このカタログをもらってもいいですか。
我可以拿这个目录吗。
```
- **（動詞て形）＋もいいですか**：在句尾加 ```か``` 表示疑问

# 可以不做……。/ 不这么做也可以

### なくてもいいです

**（動詞ない形）＋なくてもいいです**
```markdown
ボールペンで書かなくてもいいです。
不用圆珠笔填写也是可以的。
```

# 不准做……。

### てはいけません

**（動詞て形）＋はいけません**
```markdown
このノートパソコンは、使ってはいけません。
这台笔记本电脑，不可以使用。
```
- 这个句型，可以直接用```いいえ、いけない```来回答```いいですか```的疑问句
- 这个句型，不能对上级，长辈使用（因为是一个明确的禁止的句式

# 必须做……

### なければなりません

**（動詞ない形）＋なければなりません**
```markdown
ボールペンで書かなければなりません。
必须用圆珠笔写。
```
- 语气很重，不管行为者的意志如何，都要做
- ```なきゃ```简写

### ないといけません

**（動詞ない形）＋ないといけません**
```markdown
ボールペンで書かないといけません。
不用圆珠笔写是不行的。
```
- 语气中等，从行为者的意志出发，类似于，不怎么样就不可以的哦，的劝说
- ```ないと```简写

# 在……之前

### までに

**（時間名詞）＋までに**
```markdown
レポートは、木曜日までに書いてください。
报告，请在周四之前完成。（包含周四
```
- 包含该名词表示的时间点（闭区间

### まえに

**（動詞辞書形）＋まえに**
**（名詞）＋の＋まえに**
**（時間名詞）＋まえに**

```markdown
ご飯を食べるまえに、
吃饭前，

朝ご飯のまえに、
早饭前，

日曜日まえに、
周日之前，

３時まえに、
3点之前
```
- 不包含接续表示的时间点（开区间

# 颇，很，相当

### なかなか
```markdown
なかなか時間がかかる
颇费时间。

なかなか怒らない
轻易不生气。
```
- 这类词可以单独做句子成分去修饰，即，在翻译时，可以忽略，然后再加上一个程度副词即可

# 并列修饰

## 既……也……

### い形容詞（~~い~~）＋くて・な形容詞で

**い形容詞（~~い~~）＋くて**
**な形容詞で**
```markdown
このパソコンは、軽くて便利です。
このパソコンは、便利で軽いです

这台电脑，又轻便又好用。
```
- 形容词的```て```接续，是 い形容詞```去い＋くて```，な形容詞```去な＋で```

# 并列动作

## 又……又……，时而……时而……

### したり、したりする

**（動詞た形）＋り、（動詞た形）＋りする**
```markdown
水曜日は、朝礼をしたり、会議をしたりします。
周三，又开了早会，又开了组内会议。
```
- 时态由句尾的```する```决定，本身句型没有过去含义（非常要注意这一点，**动词的形变有时态含义，但是并不是所有的形变，都表示对应的时态含义**
- 不能用于表示每天做的事情，很不自然

# 多亏了……

### おかげで

**（動詞普通形）＋おかげで**
**（名詞の）＋おかげで**
```markdown
春になるおかげでたくさんの花が見えます。
幸亏春天到了，才能看到许多花儿。

先生のおかげで、日本語がうまくなりました。
多亏了老师，日语变好了。
```
- ```おかげさまで``` 更加委婉，更加尊敬

# 授受动词

## 给（为）我做……
他人 → **我**

### くれます

**（誰）は＋（私に）＋（名詞）＋を＋くれます**
**（誰）は＋（私に）＋（動詞て形）＋くれます**

```markdown
達雄さんは、私に、ケーキをくれました。
达雄给了我蛋糕。

達雄さんは、私に、電話番号を教えてくれました。
达雄告诉了我电话号码
```
- 表达受到恩惠，表达感谢。语气要十分委婉
- ```私に```可以省略，因为```くれる```出现了，一定是我受到了恩惠

## 我从……得到……（恩惠）
我 ←**获得恩惠**← 他人

### もらいます

**（私）は＋誰に・から＋（名詞）＋を＋もらいます**
**（私）は＋誰に・から＋（動詞て形）＋もらいます**

```markdown
私は、紀子ちゃんに、電話番号を教えてもらいました。
我向纪子要了电话号码。
```
- 表达受到恩惠，表达感谢。语气要十分委婉
- 这里的```に```可以替换为```から```

## 我为……做……

###  あげます

**（私）は＋誰に＋（名詞）＋を＋あげます**
**（私）は＋誰に＋（動詞て形）＋あげます**

```markdown
私は、達雄さんに、ケーキを作ってあげました。
我为达雄做蛋糕
```
- 表达**给予恩惠**，不能对关系生疏的，地位高的人用
