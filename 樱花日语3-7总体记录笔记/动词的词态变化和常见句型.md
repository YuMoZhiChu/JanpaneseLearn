# 普通形と丁寧形

- 动词，名词，形容词，都有普通型和尊敬型
- 尊敬型跟动词形变没有任何关系，任何对应动词形变，都有尊敬型（而且尊崇一个套路
- 如果标注了是，**动词普通型**接续的语法，那么就是怎样的形变都可以接，只需要改成普通型即可

# 可能動詞と対する２グループ動詞

### 見られる　｜　見える
[网上的解析1](https://www.sohu.com/a/300352973_696863)
[网上的解析2](https://jp.hjenglish.com/new/p120478/)

見える　→　没有障碍物，没有受到任何阻碍、自然地映入眼帘。
見られる　→　并非自然地映入眼帘，而是根据自己的意志选择性地看到。

```
鬼を見える。
撞见鬼了（很自然的看到

窓を開けると、富士山が見える。
打开窗，就能看到富士山。

鬼を見られる。
能看见鬼（比如开了阴阳眼

正月、日本では着物を着た女性の姿が見られます。
正月时，在日本可以看到穿和服的女性。
```

### 聞ける　｜　聞こえる

聞こえる　→　没有障碍物，没有受到任何阻碍、自然地传入耳朵。
聞ける　→　并非自然地传入耳朵，而是根据自己的意志选择性地听到。


```
ソフトで、音楽を聞ける。
可以用软件来听音乐。

トイレから子供の泣いた声が聞こえる。
厕所传来了孩子的哭声。
```

### できる

#### 能做到
```
そんなこと私にはできません。
那种事我做不到。
```

#### 完成，产生
```
宿題はぜんぶできた
作业全都做完了。
```

# て形

### ています

**（動詞て形）＋ています**
```markdown
中田さんは、会議室で、中村さんと話しています。

中田先生，在会议室，和中村先生在聊天。
```
- 这里表示的，是正在做某事。也是最简单的用法。

```markdown
中村さんは、東京に住んでいます。
中村先生，在东京住。

先生は結婚しています。
老师结婚了。

学校で、体育を教えています。
在学校教体育。

隣の町で花を売っています。
在隔壁的城市卖花。

入り口が閉まっています。
入口关闭了。

パソコンが壊れています。
电脑坏了。
```
- 动作结果的持续。（住的例子
- 动作发生了，结果一直持续进行。（结婚例子
- 一直的，不间断的做某事。（卖花，教书的例子
- 主体发生变化后，留下的结果所处的状态（入口，电脑的例子
*在这里，要记住 ています 不仅可以表示正在进行时，还可以表示其他的含义*

**知っている**
- 这个短语，表示，已经知道了
- ```知ります```，表示，快知道了
- ```知らない```，表示，不知道

### ていない

**（動詞て形）＋ていない**

```
まだ開いていない
还没开门

还没吃
```
- 通常会和修饰```まだ```一起出现

### て＋

#### 表示动作的接续

**（動詞て形）、**
**（動詞て形）＋から**

```markdown
朝、ご飯を食べて、学校にいく。
早上，吃了早饭，然后去上学

夜、テレビを見てから、寝ました。
晚上，看了电视，然后睡觉了
```
- 句子的时态，由后面的动词决定。

#### 表示动作进行的状态，方法

**（動詞1て形）+動詞2**

```markdown
ぼくは刺身に山葵をつけて食べます。
我蘸着芥末吃刺身
```

**（動詞1ない形）＋ないで+動詞2**

```markdown
ぼくは刺身に山葵をつけないで食べます。
我不蘸着芥末吃刺身
```

#### 轻微的表示原因

**（動詞て形）、**
```markdown
達雄さんの話を聞いて、がっかりしました。
听了达雄君的话，很失望。
```
- 和其他表示原因的比较，查看句意部分。

# 条件形・仮定形

### ば・なら

**（動詞仮定形・イ形容詞仮定形・ナ形容詞仮定形）**
**名詞＋なら**

```markdown
34番のバスに乗れば、地下鉄の駅に行けます。
如果坐34号公交的话，就能到地铁站了。

美味しければ、もっと食べてください。
如果好吃的话，请多吃点。

質問がなければ、これで会議は終わります。
如果没有问题的话，会议就结束吧。

好きなら、告白しろ。
如果喜欢的话，就去告白吧。

いい天気なら、ここから富士山が見えます。
如果好天气的话，在这里能看到富士山。
```
- 为了某件事情，或者某个动作的成立，对其必要的条件进行假设
- **名詞＋なら** 可以理解成名词的假定

# 受身形

### Aは＋Bに＋受身動詞

```markdown
ぼくは課長に叱られた。
我被课长骂了。
```
- A 是动作的承受者
- B 是动作的实施者

### Aは＋Bに＋～を＋受身動詞

```markdown
部長は田中さんの本を捨てました。
田中さんは部長に本を捨てられました。
田中的书被部长扔掉了。
```
- A 是动作的承受者
- B 是动作的实施者
- `を`提示所有物
- 注意，在这里，虽然只出现了人，但实际上是这个人的物品

### Aは＋受身動詞

```markdown
我が社の商品は、世界中で販売されています。
我司的商品，在世界范围中销售。
```
- 该句型的重点，在动作的承受对象

### Aは＋Bによって＋受身動詞

```markdown
この服はBGプランニングによってデザインされました。
这件衣服是BG企划设计的
```
- 一般接，创造，发现，发明，这一类的动词
