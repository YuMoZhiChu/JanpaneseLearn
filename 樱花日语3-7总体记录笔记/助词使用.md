
# は・が

### 大小主语

**（名詞1）は（名詞2）が**

```markdown
中村さんは、背が高いです。
中村先生的个子很高

京都は八つ橋が有名です。
京都的八桥很有名
```
- 在这里，用两个不同的助词，去提示我们要叙述的主语（切记，不要用```の```来连接，这样很中式

### ～＋は＋～です
```
サッカーを見るのは面白いです。
看球赛很有意思。

サッカーを見るのが面白いです。
只看球赛很有意思。
```
- 在正常情况下，用`は`，如果这个时候换成`が`，就是强调

### ～＋は＋～です
```
ピアノを引くのがすきです。
喜欢弹钢琴。

ピアノを引くのはすきです。
相比之下，我还是喜欢弹钢琴。
```
- 在正常情况下，用`が`，如果这个时候换成`は`，就是强调

### ~＋を＋～です
```
小野さんが結婚するのを知っている。
我知道小野结婚了。

小野さんが結婚するのは知っている。
小野结婚的事情，我是知道的
```
- 在正常情况下，用`を`，如果这个时候换成`は`，就是强调

### 表示数量的最低限度

**（数量）＋は**
```
どんな安いネクタイでも数万円はすると思います。
我想，领带再怎么便宜也要数万日元吧。
```
- 表示，最起码要……的数量才行

# が

### 感觉到……

**（名詞）＋がします**
```
声がします
听到声音

吐き気がします
觉得恶心

気がします
感觉到，预感到

寒気がします
可能生病了
```
- 与个人意志无关，由感觉器官捕捉到的现象

# も

### 表示强调

**（数量）＋も**
```
このネクタイ、２万円もするんですか。
这条领带，竟然要2w日元。
```
- 在数量词后面，表示语气（如果是少的话，就是竟然这么便宜

# で

### 表示手段，材料
```markdown
ペンで書く。
用钢笔写。

英語で話す。
用英语说。

この服はシルクで作られました。
这件衣服是丝绸做的。
```
- 表材料时，直接用肉眼可以判断的材料，用`で`，有一个类似的`から`,但这个是暗示加工过的
```
植物の草から作られている服もあるですね。
用植物的草做的衣服也是存在的哦。（草是加工过的
```

### 表示动作，事件发生的场所。
```markdown
学校で勉強する。
在学校学习。

彼は北京で生まれ、北京で育った。
他生在北京,长在北京。

埼玉で日本とドイツのサッカーの試合があります。
在埼玉举行日本和德国的足球比赛。
```

### 限定空间，时间的范围。
```markdown
クラスで一番の成績。
成绩全班第一。

世界で一番長い河。
世界上最长的河。
```

### 表示原因，理由。
```markdown
火事で一文なしになる。
因遭遇火灾而变得身无分文。

スピード違反で捕まる。
因超速行驶而被逮住
```

### 表示作为基准的单位量。
```markdown
三本で百円。
3根100日元。

１時間で百キロメートル進む。
一小时行进100公里。
```

# に

### 表示存在
**（名詞1）に（名詞2）がある**
```markdown
机の上に本があります。
桌子上有书。
```

### 表示目的

**（名詞）へ（動詞ます形/名詞）に行きます・来ます・帰ります**
```markdown
私は、福岡へ会議をしに行きます。
我去福冈开会。

仕事をしに来ました。
我来工作了。

あした京都のお祭りに行きます。
明天去看京都的庙会。
```
- **（動詞ます形/名詞）+ に** ： 表示目的的动词，或者是表示目的目标的名词。

### 表示到达点

**（名詞）に＋入ります・乗ります**
```markdown
会議室に入ります。
进入了会议室。

バスに乗る。
上了公交车。
```
- **（名詞）に＋入ります・乗ります** ： 表示到达点，进去某个地方。

### 表示用途

**（名詞）に＋動詞**

```markdown
この印刷機は、２０枚以上刷るのに使います。
この印刷機は、２０枚以上の印刷に使います。
这台印刷器是为了20张以上的印刷使用的。
```
- `するの`转换成了名词

### 表示原因，因为……而

**（名詞）に**

```markdown
ずば抜けて良い営業成績に満足しています。
我对优异的营业成绩感到满意。
```

- 相当于汉语的，因为/为了……而

### 表示动作，作为的目的

 **（動詞ます形）+ に** 

```
健康を維持するには運動は一番です。
为了保持健康，运动是最好的选择。
```

- 相当于汉语的 **为了……** 的意思

### 表示原因，手段，方法

#### によって（は）・により・による

 **名詞＋によって（は）・により・による** 

```
誰が誘ってくれるかによって、デートするかしないかを決める。
看邀请人是谁，再决定是否去赴约。

この事故による影響で、
由于这次事故的影响，
```

#### によると、によれば

```
天気予报によると、明日雨だそうです。
据天气预报说，明天会下雨。

新闻によると、彼は自杀したそうです。
据报纸报道，他自杀了。
```
- 接句子的形式
- 两者意思完全一样。如果一定要区别,一般口语经常说`によると`,书面的话用`によれば`。

### 表示关于……

#### についての・につき

**名詞＋についての・につき**

```markdown
中国ビジネスについての本がよく売れています。
关于中国商务的书卖得很好。

その件につき質問させてください。
关于那件事请让我提问。
```

- 表示提出话题，并对其进行阐述。**关于~**，其中``是比较郑重的方法
- `につき`还可以表示，原因理由，或者，每当
```
この犬、凶暴につき
这只狗很凶

一人につき千円
每人一千日元
```

### 表示对……来说

#### にとっては・にとっての

```
このプロジェクトは、僕にとっては重要なんです。
这个项目对我来说很重要。

私たちにとっての問題は、ファッションに対する価値観が違うことです。
对我们来说的问题是对时尚的价值观不同。
```

- 表示站在某个角度的评价或判断

# を

### 表示离开点

**（名詞）を＋出ます・降ります**
```markdown
会議室を出ます。
走出会议室。

バスを降る。
下了公交车。
```
- **（名詞）を＋出ます・降ります** ： 表示离开点，离开某个地方。

# と

### 一就会，一就要

**（動詞辞書形）＋と**

```markdown
この道をまっすぐ行くと、駅があります。
沿着这条路直走，就到车站了。

春になると花が咲く
一到春天花就会开
```
- 后半句不能使用意志，希望，劝诱，依赖等表达，只能是：
- - 既定事实
- - 自然现象
- 句末应该用现在时态。

#### 只要做……就可以了

**（動詞辞書形）＋といい**

```markdown
このボタンを押すといいです。
只要按这个按钮就可以了。

明日は晴れるといいですね。
明天如果天气好就好了。
```
- 这个句型是表达，只要做什么就可以了。
- 也可以表达愿望，注意在后面加```ね```

#### 不做……是不行的哦

**（動詞ない形）＋ないとだめ**

```markdown
このボタンを押さないとだめです。
只要不按这个按钮是不行的哦。
```
- 这个同质于：
```
ないといけない
ないといけません
```

### 提示内容，引语

#### 读作……

**＋と読みます**

```markdown
あの漢字は何と読むんですか？
这个汉子怎么读啊？

あれは「禁煙」と読みます。
这个汉子读作禁烟。
```

#### 这个是……的意思

**＋という意味です**
**～とは、（名詞）＋の意味です**
**～とは、（名詞・句）＋の意味です**

```markdown
これは、「自転車は入るな」という意味です。
这是自行车不准进来的意思。

アテンドとは、「送迎」の意味です。
アテンドとは、「送迎」という意味です。
attend是接送的意思。
```

#### 转达

**＋と伝えてください**
**＋と伝えてくださいませんか**
**＋と伝えてくださいただけませんか**

```markdown
あした、「中村は出張だ」と伝えてください。
明天请帮忙转达中村出差了。
```
