# 顺接

### たら

#### 如果…就（假定顺接

**（動詞た形）＋ら**
**い形容詞（~~い~~）＋かったら**
**な形容詞・名詞＋だったら**
```markdown
雨が降ったら、バーベキューは中止です。
雨だったら、バーベキューは中止です。
如果下雨的话，就中止野餐。
```
- 可以在句首加上```もし```来强调语气

### それに

#### 再加上、并且（第三个原因的顺接

```
晴れたし、それに風もない
天很晴朗而且没有风。

この料理まずいし、それに高い
这个饭又难吃又贵。
```

### それに

#### 因此，所以

```
彼は予備校へ通った。それで成績がよくなった。
他在补习学校学习，所以成绩好了。
```

#### 那么，后来。

```
それでどうした？
后来怎样了？
```

# 逆接

### でも

#### 但是

```markdown
あの人に何度も手紙を出してみたわ。でも、一度も返事をくれなかったの。
我给他去过好多次信。可是，他一次也没给我回信。
```

#### 即使……也、不管……都

**（動詞て形）＋も**
**い形容詞（~~い~~）＋くても**
**な形容詞・名詞＋でも**
```markdown
雨が降っても、バーベキューをしましょう。
即使下雨，也要野餐。

いくら寒くても、スカートを履きたい。
不管有多冷，也想穿短裙
```
- 可以在句首加上```いくら```来强调语气

### けど

#### 但是，其实

```markdown
あの人は丈夫そうにみえる。けど、よく病気をする。
那个人看起来很壮实，但（实际上）常常生病。
```

# 变成

### なる

**い形容詞（~~い~~）＋くなります**
**な形容詞＋になります**
**名詞＋になります**
```markdown
熱くなる。
变热

きれいになる。
变得漂亮
```

### ようになりました

**動詞辞書形＋ようになりました**
```markdown
新しい地下鉄で通勤できるようなりました。
因为新的地铁，所以能够通勤了。
```

**動詞ない形＋なくなりました**
```markdown
少し太って、好きな服が着られなくなりました。
稍稍变胖了，喜欢的衣服穿不了了。
```

- 表示能力，状况，习惯的变化，一般使用可能动词比较合适
- 不能接表示变化的动词（如果这类动词，直接用即可
- 一般是```なりました```，因为已经变化了

# 习惯，坚持（不）做……

### ように

**（動詞辞書形）＋ようにしています**
```markdown
毎週運動するようにしています。
```

**（動詞ない形）＋ないようにしています**


# 请习惯，坚持，尽量（不）做……

### ように

**（動詞辞書形）＋ようにしてください**
```markdown
もっと睡眠時間をとるようにしてください。
请保证你的睡眠时间。
```

**（動詞ない形）＋ないようにしてください**
```markdown
タバコを吸わないようにしてください。
请你戒烟
```

- 不能用于即时的请求，比如```本を貸すようにしてください```（不可能坚持去借给你吧

# 为了（不）能够实现……这一目标

### ように

**（動詞辞書形）＋ように、**
```markdown
仕事がうまくいくように、毎朝新聞を読んでいます。
为了保证能很好的工作，每天早上都读新闻。
```

**（動詞ない形）＋ないように、**
```markdown
先生から聞いたことを忘れないようにメモをとります。
为了记住老师讲的东西，做笔记
```

- 前面用非意志性的动词（前面应该是比较客观的事物，不能表示为了一些个人想法的目标，比如进大学

# 做……吧。

### ましょうか。

**（動詞ます形）＋ましょうか**
```markdown
明日も来ましょうか。
ええ、１０時に来てください。

我明天也来吧？
恩，请10点钟过来。

傘を貨しましょうか。
すみません。お願いします。

我把伞借给你吧。
谢谢，我借一下。

お腹がペコペコだよ。
じゃ、ご飯を食べに行きましょか。

肚子饿了啦。
那，我们一起去吃饭吧。

```
- 注意3种不同的使用场景，可以用于自身，或者一起的提议（一定要有自己

# 请……。

### てください

**（動詞て形）＋ください**
```markdown
ちゅっと待ってください。
请稍等一下下。

駅の読み方を教えてください。
请告诉我车站的读法。
```
- **すみませんが、…… +（動詞て形）＋ください**：在句首加 ```すみませんが``` 会显得更加委婉

# 请不要……。

### ないでください

**（動詞ない形）＋ないでください**
```markdown
このパソコンは、使わなくてください。
请不要使用这台电脑。
```

# 可以做……。

### てもいいです

**（動詞て形）＋もいいです**
```markdown
パソコンを使ってもいいです。
可以使用电脑哦。

このカタログをもらってもいいですか。
我可以拿这个目录吗。
```
- **（動詞て形）＋もいいですか**：在句尾加 ```か``` 表示疑问

# 可以不做……。/ 不这么做也可以

### なくてもいいです

**（動詞ない形）＋なくてもいいです**
```markdown
ボールペンで書かなくてもいいです。
不用圆珠笔填写也是可以的。
```

# 不准做……。

### てはいけません

**（動詞て形）＋はいけません**
```markdown
このノートパソコンは、使ってはいけません。
这台笔记本电脑，不可以使用。
```
- 这个句型，可以直接用```いいえ、いけない```来回答```いいですか```的疑问句
- 这个句型，不能对上级，长辈使用（因为是一个明确的禁止的句式

# 必须做……

### なければなりません

**（動詞ない形）＋なければなりません**
```markdown
ボールペンで書かなければなりません。
必须用圆珠笔写。
```
- 语气很重，不管行为者的意志如何，都要做
- ```なきゃ```简写

### ないといけません

**（動詞ない形）＋ないといけません**
```markdown
ボールペンで書かないといけません。
不用圆珠笔写是不行的。
```
- 语气中等，从行为者的意志出发，类似于，不怎么样就不可以的哦，的劝说
- ```ないと```简写

# 授受动词

## 给（为）我做……
他人 → **我**

### くれます

**（誰）は＋（私に）＋（名詞）＋を＋くれます**
**（誰）は＋（私に）＋（動詞て形）＋くれます**

```markdown
達雄さんは、私に、ケーキをくれました。
达雄给了我蛋糕。

達雄さんは、私に、電話番号を教えてくれました。
达雄告诉了我电话号码
```
- 表达受到恩惠，表达感谢。语气要十分委婉
- ```私に```可以省略，因为```くれる```出现了，一定是我受到了恩惠

## 我从……得到……（恩惠）
我 ←**获得恩惠**← 他人

### もらいます

**（私）は＋誰に・から＋（名詞）＋を＋もらいます**
**（私）は＋誰に・から＋（動詞て形）＋もらいます**

```markdown
私は、紀子ちゃんに、電話番号を教えてもらいました。
我向纪子要了电话号码。
```
- 表达受到恩惠，表达感谢。语气要十分委婉
- 这里的```に```可以替换为```から```

## 我为……做……

###  あげます

**（私）は＋誰に＋（名詞）＋を＋あげます**
**（私）は＋誰に＋（動詞て形）＋あげます**

```markdown
私は、達雄さんに、ケーキを作ってあげました。
我为达雄做蛋糕
```
- 表达**给予恩惠**，不能对关系生疏的，地位高的人用

# 在……之前

### までに

**（時間名詞）＋までに**
```markdown
レポートは、木曜日までに書いてください。
报告，请在周四之前完成。（包含周四
```
- 包含该名词表示的时间点（闭区间

### まえに

**（動詞辞書形）＋まえに**
**（名詞）＋の＋まえに**
**（時間名詞）＋まえに**

```markdown
ご飯を食べるまえに、
吃饭前，

朝ご飯のまえに、
早饭前，

日曜日まえに、
周日之前，

３時まえに、
3点之前
```
- 不包含接续表示的时间点（开区间

# 在……之后

### あとで

**（動詞た形）＋あとで**
```markdown
書類を作ったあとで会議に参加する。
```


# 越……越……

**（動詞仮定形）＋ば＋（同じ動詞の辞書形）＋ほど**

# 并列举例

## ~啦~啦

### とか

**（名詞）＋とか**
**（動詞、い形容詞、な形容詞）普通形＋とか**

```markdown

```
- 名词的话，普通形```英語の文法だとか```来并列确实很膈应


# 并列修饰

## 既……也……

### い形容詞（~~い~~）＋くて・な形容詞で

**い形容詞（~~い~~）＋くて**
**な形容詞で**
```markdown
このパソコンは、軽くて便利です。
このパソコンは、便利で軽いです

这台电脑，又轻便又好用。
```
- 形容词的```て```接续，是 い形容詞```去い＋くて```，な形容詞```去な＋で```

### し

**（動詞、い形容詞、な形容詞、名詞）普通形＋し**
```markdown
今日は遅刻するし、財布を忘れるし、まいったなあ。
今天迟到了，钱包也忘记带了，真的太惨了。

このレストランは安いし、美味しいし、いいですね。
这家店又便宜有好吃。

英語も上手だし、中国語も上手だし、すごいですね。
既擅长英语，也擅长中文，太厉害了。
```
- 句中会出现```も```来表达语气。


# 并列动作

## 又……又……，时而……时而……

### したり、したりする

**（動詞た形）＋り、（動詞た形）＋りする**
```markdown
水曜日は、朝礼をしたり、会議をしたりします。
周三，又开了早会，又开了组内会议。
```
- 时态由句尾的```する```决定，本身句型没有过去含义（非常要注意这一点，**动词的形变有时态含义，但是并不是所有的形变，都表示对应的时态含义**
- 不能用于表示每天做的事情，很不自然

## 一边做……，一边做……

### ながら

**（動詞ます形）＋ながら、**
```markdown
毎朝コーヒーを飲みながら、新聞を読みます。
每天早上一边喝咖啡，一边看新闻。

アルバイトをしながら、大学に行っています。
边打工，边上大学。
```
- 用于持续时间较长的事情时，注意使用持续性动词，和时态（上大学
- 强调的是后面接续的动作

# 感慨

### てしまいます

**（動詞て形）＋てしまいます**

#### 很遗憾，很困扰，失败了
```markdown
傘をバスに忘れてしまいました。
把雨伞落在公交车上了。
```

#### 终于做完了！
```markdown
この仕事をやってしまいました。
这件工作终于做完啦。
```
- 在不需要强调的时候，使用这个句型，会很不自然

# 预先做好……

### ておきます

**（動詞て形）＋ておきます**

```markdown
ホテルを予約しておきます。
提前预定酒店。

母は毎朝六時前に、朝食を作っておきます、
妈妈每天早上6点前，起来做早餐
```
- 为了某种目的或事物的准备而做的某事

# 已经做好了……

### てあります

**（動詞て形）＋てあります**

```markdown
ホテルは予約してあります。
预定好了酒店。

そして、子供が起きる時、朝食はもう作ってあります。
孩子们起来的时候，早餐已经做好了
```
- 这个状态的停留，多用于```他動詞```,因为```自動詞```用的是```ています```
- 可以留意到，这里使用的助词，不是表动宾关系的```を```，而是表示提示主语的```は```

# 打算做……

### つもりです

**（動詞辞書形）＋つもりです**
**（動詞ない形）＋ないつもりです**

```markdown
今日は残業をするつもりです。明日は残業をしないつもりです。
今天打算加班，明天打算不加班。
```
- 用于表达**自己**的意志，计划。我打算....

### 予定です

**（動詞辞書形）＋予定です**

```markdown
週末は桜ちゃんとデートする予定です。
周末打算和小樱约会。
```
- 书面的表达，客观的计划

# 推测，推量

## 我觉得……

### と思う

**（動詞普通形）＋と思う**
```markdown

```

### と思っています

**（動詞意向形）＋と思う、と思っている**

- 这个句型，多了已经下定决心做某事的意思。

# 多亏了……

### おかげで

**（動詞普通形）＋おかげで**
**（名詞の）＋おかげで**
```markdown
春になるおかげでたくさんの花が見えます。

多亏了老师，日语变好了。
```
- ```おかげさまで``` 更加委婉，更加尊敬

# （相比之下）还是做……比较好

### ほうがいい

**（動詞た形）＋ほうがいい**
**（名詞の）＋ほうがいい**
```markdown

```
- ```た``` 并不表示过去，而是表示动作的完成，就你做完某件事会比价好

**（動詞ない形）＋ないほうがいい**

```markdown

```

# 在/正在……半途上

### 途中で

**（動詞て形＋いる）＋途中で**
**（動詞辞書形）＋途中で**
**（名詞の）＋途中で**
```markdown
実は、途中で事故があって、荷物が遅れてしまいました。
实际上，中途遭遇了事故，货物延期了。
```
- 这里的接续，实际上是，动词和名词的一类接续，但是这个不同的是，表示途中，只能用`動詞辞書形`或者`動詞て形＋いる`
